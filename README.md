# CommandPatternDrawingApp
CommandPatternDrawingApp is a C# console application that demonstrates the Command Design Pattern. Users can draw circles and rectangles, view shapes, and perform Undo/Redo operations. The project showcases object-oriented design, loose coupling, and command history management using .NET 8 and VS Code.
